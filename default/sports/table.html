Table Tennis

Table tennis, also known as ping pong, is a sport in which two or four players hit a lightweight ball back and forth across a table using a small, round bat. The game takes place on a hard table divided by a net. Except for the initial serve, players must allow a ball played toward them only one bounce on their side of the table and must return it so that it bounces on the opposite side
